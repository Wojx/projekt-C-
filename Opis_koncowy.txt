Wszystkie założenia projektu zostały spełnione. Głównymi składowymi aplikacji są WPF, Entity Framework oraz Google Maps API v3.
Obiekt przeglądarki Web Browser odpowiada za wyświetlenie strony html zawierającej skrypty JS generujące mape. Metoda Invoke() 
klasyWeb Browser uruchamia metody po stronie JS. Klasa HTMLConnection steruje komunikacją JS -> C#. Metoda window.external. uruchamia
kod po stronie C#. Aplikacja komunikuje się z lokalną bazą danych poprzez Entity Framework (sposób code first, na podstawie klas encyjnych jest
generowana baza danych). W bazie są zapisane dane logowania zarejestrowanych użytkowników oraz wyszukane przez nich trasy.

Szczegóły technologii Google Maps API v3.
W projekcie zostały wykorzystane:
	-podstawowe Google Maps API, służące za generowanie i obsługę mapy
	-geocoder API służące do przetwarzania adresów, nazw i wyszukiwania ich na mapie
	-places API służące do wyszukiwania szczegółowych danych o znalezionych miejscach (zdjęcia, adres, opis)
	-direction API służące do wyznaczania i wyświetlania trasy pomiędzy dwoma punktami
